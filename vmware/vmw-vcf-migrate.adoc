= Scopri come convertire un ambiente vSphere con datastore esistenti in VCF
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
La conversione di un ambiente vSphere con datastore esistenti in esecuzione su ONTAP in un ambiente VMware Cloud Foundation (VCF) implica l'integrazione dell'infrastruttura attuale in una moderna architettura cloud privata.

Questo processo sfrutta la flessibilit√† dell'archiviazione ONTAP per un accesso e una gestione dei dati senza interruzioni.  Una volta stabilito il dominio di gestione VCF, gli amministratori possono importare in modo efficiente altri ambienti vSphere nell'ecosistema VCF.  Questa integrazione migliora l'utilizzo delle risorse, semplifica la gestione del cloud privato e garantisce una transizione fluida con un'interruzione minima dei carichi di lavoro esistenti.
