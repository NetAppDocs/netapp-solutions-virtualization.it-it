Il SnapCenter Plug-in for VMware vSphere consente operazioni di backup e ripristino rapide e coerenti con le VM per VM, datastore e file VMDK.  Questo plug-in VMware si integra con SnapCenter Server per supportare il backup e il ripristino basati sulle applicazioni per i plug-in specifici delle applicazioni SnapCenter .
